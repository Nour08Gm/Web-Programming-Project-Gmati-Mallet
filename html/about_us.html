Project Overview

This website was developed as part of the TI402 Web Development course at Efrei.
Our mission was to create a professional showcase for the Computer Science Department.

Technical Specifications

  • Developed using HTML5, CSS3, and Vanilla JavaScript.
  • Responsive Design: Optimized for mobile, tablet, and desktop views.
  • W3C Compliant: Validated code for high-quality standards.
  • Interactive Features: Dynamic navigation and UI elements powered by JS.
